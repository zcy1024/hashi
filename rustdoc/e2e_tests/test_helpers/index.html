Shared helpers used by e2e test modules.

Test modules across this crate (e2e_flow, upgrade_tests, …) all need the same boilerplate to drive a localnet: init tracing, look up an hBTC balance, wait for a DepositConfirmedEvent, deposit-and-wait, etc. Define them here once and import from each test module.
