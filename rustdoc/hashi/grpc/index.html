MPC_PROTOCOL_METADATA_KEY
Metadata key used to tag outbound MPC RPCs with the originating protocol (e.g. DKG, signing). Set by mpc::rpc::p2p_channel::RpcP2PChannel and read by metrics middleware to label per-protocol traffic.
