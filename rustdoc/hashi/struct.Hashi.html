pub fn get_onchain_mpc_pubkey(&self) -> Result<XOnlyPublicKey>

Read the MPC public key from on-chain state.

Unlike get_hashi_pubkey, this does not depend on the node’s local DKG completion channel. The on-chain key is set during end_reconfig event processing, so it is guaranteed to be present once the initial committee has formed (i.e., after HashiNetworkBuilder::build() returns).

pub fn build_unsigned_withdrawal_tx( &self, selected_utxos: &[Utxo], outputs: &[OutputUtxo], ) -> Result<Transaction>

Build an unsigned Bitcoin transaction for a withdrawal. This is used both by the leader when initially crafting the tx, and by validators when verifying that a proposed WithdrawalTxCommitment produces the expected txid.

Source

pub async fn build_withdrawal_tx_commitment( &self, requests: &[WithdrawalRequest], ) -> Result<WithdrawalTxCommitment, WithdrawalCommitmentError>

Build a withdrawal commitment for a batch of approved requests: select UTXOs using the batching-aware coin selection algorithm, build the unsigned BTC tx, and return a WithdrawalTxCommitment covering the selected requests.
