Sui RPC client for interacting with the Hashi on-chain state

This module provides a client for reading Hashi state and building/executing proposal-related transactions.

Uses OnchainState from the hashi crate for reading on-chain data, and SuiTxExecutor for transaction execution when a keypair is configured.
